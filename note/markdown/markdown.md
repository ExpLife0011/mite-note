### 2  语法高亮
    用 `` 插入代码
    `这是代码块`
    用 ```json 代码块 ```插入语法高亮代码
```json
{
    "key":value,
    "key2":valie,
}
```
    用 ```c 代码块 ``` 插入C代码
```c
    int main()
    {
        int a = 0;
        return 0;
    }
```

### 4 粗体与斜体
用两个 ** 号包含一段粗体内容

用一个*号码包含一段斜体

**这里是一段粗体**

*这里是一段斜体*
